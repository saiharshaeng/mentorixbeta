## Running Mentorix
1. cd to the mentorix folder
2. node src/server.js
3. Open http://localhost:8080
NEVER double-click index.html directly.

# Mentorix AI Learning Ecosystem
Mentorix is a premium, playful, and adaptive AI learning platform designed to help students master complex subjects and prepare for competitive exams.

### Features
* **Tio AI Mentor:** Empathic, witty, and contextual learning companion.
* **Smart Study Chapters:** Adaptive 9-step syllabus progression with notes, checks, and assessments.
* **CBT Test Center:** Full mock exam environment supporting JEE, NEET, and other global exams with adaptive question banks and real-time grading.
* **Skill Recovery Center:** Personalized diagnostics that identify and patch cognitive weaknesses.
